Конспект

Основы

Программа на языке Python выполняется последовательно (построчно). Так, следующая программа поочерёдно выведет Hello world, Python 3 и 42:

print("Hello world!")	# print() используется для вывода данных
print("Python 3")	# Символ решётки создаёт комментарий — текст,
print(42)		# который игнорируется программой

Выражение — участок кода, вычисляющийся в значение. Например, выражение 5 + 10 вычисляется в 15. Простейшим выражением является константа — значение, не требующее вычисления: 5, -10, 1.5, "Hello" и т. д.

print(5)		# Вывод константы 5
print(5 + 10 - 2)	# Вывод результата вычисленного выражения (13)

Переменная

Переменная — имя, с помощью которого можно обратиться к данным, сохранённым в памяти компьютера.

В Python переменная создаётся записью вида
<название_переменной> = <значение переменной>
Например, запись x = 5 создаёт переменную x со значением 5.

В переменную можно записать результат какого-то выражения:

x = 11		# Записать в переменную x число 11
y = 5 + 10	# Записать в переменную y сумму 5 и 10

И в дальнейшем использовать:

x = 20
y = 15
print(x)	# Вывести значение переменной x (20)
print(y + 2)	# Прибавить 2 к значению переменной y и вывести (17)
z = x + y	# Записать в переменную z сумму x и y
print(z)	# Вывести значение переменной z (35)

Переменные можно перезаписывать:

x = 5
print(x)	# 5
x = 7
print(x)	# 7

Переменные нельзя использовать до создания:

print(x)	# Ошибка — нет переменной с именем 'x'
x = 5

Но можно использовать при перезаписи:

x = 10
print(x)	# 10
x = x + 1
print(x)	# 11

Базовые типы данных

У любой переменной есть тип данных. Он определяет возможные значения переменной и доступные операции. Язык Python предоставляет четыре основные типа данных:

int float str bool
Тип Целое число Вещественное число Строка Логическое значение
Константа 12345 12345.0 “12345” True или False

int — целое число

Целое число — простейший тип данных, аналогичный математическому множеству целых чисел (Z).

Константа состоит из символов от 0 до 9, может иметь знак:
0, 1, 254, 3E3, -250, +400

Возможна запись константы в других системах счисления:
0xF — 15 в шестнадцатеричной (префикс 0x)
0o77 — 63 в восьмеричной (префикс 0o)
0b101 — 5 в двоичной (префикс 0b)

Функция int(x) предназначена для преобразования значений из других типов:

print(int(1))		# 1
print(int(1.0))		# 1
print(int(1.5))		# 1 (дробная часть отбрасывается)
print(int("1"))		# 1
print(int("abc"))	# Ошибка
print(int(True))	# 1
print(int(False))	# 0

float — вещественное число

Вещественное число (число с плавающей запятой) — числовой тип данных, аналогичный математическому множеству рациональных чисел (Q).

Константа содержит числа от 0 до 9 и обязательную точку, может иметь знак:
0.0, .0, 0., 0.0E0, -5., .525, 5125E-2

Функция float(x) предназначена для преобразования значений из других типов:

float(1.0)	# 1.0
float(1)	# 1.0
float("1")	# 1.0
float("1.5")	# 1.5
float("abc")	# Ошибка
float(True)	# 1.0
float(False)	# 0.0

str — строка

Строка — это последовательность символов произвольной длины. Строка не является числом, даже если все составляющие её символы являются цифрами. Так, со строкой невозможны арифметические операции без использования ранее рассмотренных функций преобразования в один из числовых типов — int или float.

Константа начинается и заканчивается на символе кавычки (одинарной или двойной):

"Hello", "Hi", "125", 'Python', 'I like "PyCharm"', "Кириллица", "🙉"

Функция str(x) предназначена для преобразования значений из других типов:

str("abc")	# "abc"
str(10)		# "10"
str(0b101)	# "-5"
str(10.5)	# "10.5"
str(True)	# "True"
str(False)	# "False"

bool — логическое значение

Логический тип данных аналогичен используемым в алгебре логики. Имеет лишь два возможных значения — True (истина, 1) и False (ложь, 0)

Функция bool(x) предназначена для преобразования значений из других типов:

bool(0)		# False
bool(1)		# True
bool(50)	# True (любое число, кроме ноля — True)
bool(0.0)	# False
bool(0.5)	# True
bool("")	# False
bool("abc")	# True
bool("False")	# True (любая непустая строка — True)

Арифметические операции

Арифметические операции возможны между двумя значениями числового типа. Результат арифметической операции имеет целый тип (int), если оба числа также имеют целый тип.

Основные арифметические операции:
+ — сложение двух чисел:

2 + 5		# 7
5 + 1.0		# 6.0
5 + 2.5		# 7.5

- — вычитание двух чисел:

5 - 1		# 4
5 - 1.0		# 4.0
2 - 5		# -3
2 - 3.5		# -1.5

* — умножение двух чисел:

2 * 2		# 4
2 * 2.0		# 4.0
5 * 0.5		# 2.5

/ — деление двух чисел, в результате всегда вещественное число:

5 / 2		# 2.5
2 / 2		# 1.0
6 / 0.5		# 12.0

// — целочисленное деление двух чисел, в результате может быть как целое, так и вещественное число без дробной части:

5 // 2		# 2
2 // 2		# 1
10 // 3.0	# 3.0
6 // 0.5	# 12.0
6 // 0.52	# 11.0 (хотя 6 / 52 -> 11.538461)

% — получение остатка от деления двух чисел:

2 % 2		# 0
5 % 2		# 1
21 % 8		# 5
21 % 9.0	# 3.0
6 % 0.5		# 0.0

** — возведение в степень:

5 ** 2		# 25
5 ** 0.5	# 5.0 (квадратный корень)
2.0 ** 15	# 32768.0

abs(x) — получение модуля числа

abs(5)		# 5
abs(-5)		# 5
abs(0.5)	# 0.5
abs(-0.5)	# 0.5

Функция также может быть выражением:

print(5*abs(-10))	# 50

Операции над строками

+ — сложение двух строк. Правая строка добавляется в конец левой:

"1" + "1"	# "11"
"1" + 1		# Ошибка — нельзя складывать строку с числом
"a" + "b"	# "ab"
"a" + "b" + "c" # "abc"

* — умножение строки:

"1" * 5		# "11111"
"AB" * 5	# "ABABABABAB"
5 * "AB"	# "ABABABABAB"
"5" * "5"	# Ошибка — нельзя умножать строку на строку

Операции над логическими значениями

and — логическое умножение (и):

False and False	# False
False and True	# False
True and False	# False
True and True	# True

or — логическое сложение (или):

False or False	# False
False or True	# True
True or False	# True
True or True	# True

not — логическое отрицание:

not False	# True
not True	# False

Пример:

x = True
y = False
z = False
print(x and not (y or not z))

# Последовательно преобразуем выражение:
# True and not(False or not False)) - подставили значения из переменных
# True and not(False or True))
# True and not True
# True and False
# False - программа выведет False!

Операции сравнения

== — равенство. True, если значения идентичны, иначе False:

5 == 5		# True
2 == 3		# False
5.2 == 5.2	# True
1.5 == 1.0	# False
5.0 == 5	# True (вещественное число без дробной части равно целому)
0b11 == 3	# True
"3" == 3	# False (строка не равна числу)

!= — обратная равенству операция. True, если значения различаются, иначе False. x != y равноценно not x == y

> — больше:

5 > 2		# True
5 > 5		# False
5 > 5.0		# False
5 > "2"		# Ошибка
"b" > "a"	# True ("b" находится позже по алфавиту, чем "a")
"900" > "1000"	# True
"Ад" > "Берёза" # False (Алфавитный порядок)

>= — больше или равно. x >= y равноценно x > y or x == y

5 >= 2		# True
5 >= 5		# True
5 >= 5.0	# True
5 >= "2"	# Ошибка
...

< — меньше (см. больше)
<= — меньше или равно (см. больше или равно)

Базовые функции

print(x) — вывести x в терминал:

print("Hello!")		# На экране отобразится: Hello!
print("123")		# На экране отобразится: 123
print(123)		# На экране отобразится: 123
print(123.0)		# На экране отобразится: 123.0
print(5 * 5)		# На экране отобразится: 25
print(True)		# На экране отобразится: True

x = 5
print("x равен: ", x)	# Можно выводить несколько значений через запятую

input() — получить значение (строку) от пользователя. Программа останавливает выполнение до нажатия клавиши Enter:

print(input()*2)	# Программа выведет ввод пользователя, повторённый дважды
print(int(input())*2)	# Программа выведет ввод пользователя, умноженный на два
x = input()		# В переменную x будет сохранён ввод пользователя (в виде строки)
x = input("Введите значение x: ") # Можно добавить пояснение

len(x) — получить длину строки x в виде целого числа:

len("Hi!")		# 3
len("     ")		# 5
len("")			# 0

Условный оператор

Условный оператор (или оператор ветвления) определяет поведение программы в зависимости от некоторого логического значения.

if (“если”)

В языке Python оператор ветвления имеет следующий вид:

if <УСЛОВИЕ>:
    <ПОСЛЕДОВАТЕЛЬНОСТЬ КОМАНД>

Где условие — некоторое логическое значение, будь то выражение, переменная или константа (True или False). Все строки кода, находящиеся после двоеточия и имеющие отступ, будут выполнены только в том случае, если <УСЛОВИЕ> — True. Например:

if True:
    print("Будет напечатано!")
if False:
    print("Не будет напечатано!")

Попробуйте запустить следующий код:

x = int(input('Введите целое число: '))
if x > 0:
    print("Число положительно")
if x == 0:
    print("Число равно нулю")
if x < 0:
    print("Число меньше нуля")

Можно создавать вложенные условные операторы:

num = input('Введите количество купленного товара: ')
if num > 0:
    print("Заказ оформлен")
    if num > 10:
        print("Бесплатная доставка!")

else (“иначе”)

С применением else условный оператор принимает следующий вид:

if <УСЛОВИЕ>:
    <ПОСЛЕДОВАТЕЛЬНОСТЬ КОМАНД>
else:
    <ПОСЛЕДОВАТЕЛЬНОСТЬ КОМАНД>

Первая последовательность команд выполнится в том случае, если <условие> — True, а вторая, если False. Например:

if True:
    print("Будет напечатано!")
else:
    print("Не будет напечатано!"
if False:
    print("Не будет напечатано!")
else:
    print("Будет напечатано!")

password = input("Введите пароль: ")
if password == "qwerty":
    print("Добро пожаловать")
else:
    print("Доступ запрещён")

elif (“иначе если”)

elif — сокращение от else if, т.е. иначе если. Условный оператор принимает следующий вид:

if <УСЛОВИЕ 1>:
    <ПОСЛЕДОВАТЕЛЬНОСТЬ КОМАНД 1>
elif <УСЛОВИЕ 2>:
    <ПОСЛЕДОВАТЕЛЬНОСТЬ КОМАНД 2>
...
else:
    <ПОСЛЕДОВАТЕЛЬНОСТЬ КОМАНД N>

<УСЛОВИЕ 2> будет вычислено и проверено только в том случае, если <УСЛОВИЕ 1> — False, а <ПОСЛЕДОВАТЕЛЬНОСТЬ КОМАНД N> будет выполнена, если все предыдущие условия — False.

Так, следующий код:

answer = input("Перезагрузить компьютер? [ДА/НЕТ] ")
if answer == "ДА":
    print("Rebooting...")
if answer == "НЕТ":
    print("Перезагрузка отменена")
if answer != "ДА" and answer != "НЕТ":
    print("Неверный ввод")

Можно упростить:

answer = input("Перезагрузить компьютер? [ДА/НЕТ] ")
if answer == "ДА":
    print("Rebooting...")
elif answer == "НЕТ":
    print("Перезагрузка отменена")
else:
    print("Неверный ввод")
